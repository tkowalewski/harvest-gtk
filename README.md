harvest-gtk
===

version: 0.1.0

## Requirements
- valac-0.18
- gee-1.0
- gtk+-3.0
- libxml-2.0
- libsoup-2.4

## Installation

	$ git clone git://github.com/tkowalewski/harvest-gtk.git
	$ cd harvest-gtk
	$ sudo make install
